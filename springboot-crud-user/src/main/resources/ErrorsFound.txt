Push It to GitHub
1. Open Project in IntelliJ
Extract the ZIP and open the folder in IntelliJ IDEA.

IntelliJ will auto-import the Maven project.

2. Create Git Repository
Go to VCS > Enable Version Control Integration → Select Git

3. Commit the Project
Go to VCS > Commit...

Write a commit message like: Initial commit

Click Commit (or Commit and Push if GitHub is connected)

4. Create GitHub Repository
Go to GitHub → Click New → Enter repo name (e.g., springboot-crud-user) → Create repo

5. Push to GitHub
In IntelliJ terminal:

bash
Copy
Edit
git remote add origin https://github.com/YOUR_USERNAME/springboot-crud-user.git
